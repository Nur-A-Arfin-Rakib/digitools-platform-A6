# 🚀 DigiTools Platform

A modern digital tools marketplace built with React.js where users can browse, buy, and manage AI-powered productivity tools.

## 🛠️ Technologies Used

- **React.js** — Component-based UI
- **Tailwind CSS** — Utility-first styling
- **Vite** — Fast build tool
- **React-Toastify** — Toast notifications

## ✨ 3 Key Features

1. **🛒 Cart System** — Add/remove products, view total price, and checkout with toast notifications
2. **🔄 Product/Cart Toggle** — Instantly switch between browsing products and managing your cart
3. **📱 Fully Responsive** — Works beautifully on mobile, tablet, and desktop
